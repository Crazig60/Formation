D’un naturel curieux des dernières innovations techniques, le développeur informatique participe à de nombreux projets au sein de son entreprise. Du développement d’un site internet à la réalisation de logiciels ou encore d’applications, sa journée est rythmée par les touches des claviers qu’il utilise.

Le métier de développeur web exige une grande rigueur. De solides connaissances dans ce domaine vous ouvriront donc de nombreuses portes.

Dans le cadre de cette formation, vous avez la possibilité de participer à une classe virtuelle (par internet) avec votre professeur. Vous pourrez également profiter d'un e-entretien individuel avec le professeur responsable de votre formation.

Le programme de votre formation développeur web à distance : Fournis avec votre formation : tous les produits de la gamme Adobe (compatibles PC/MAC) et tous les services web associés. (Licence valable 1 an à partir de la date de réception. Reconduction possible. Nous contacter).

Votre formation comporte au total 22 tomes :

-  Programmation HTML - les bases (partie 1)
-  Programmation HTML - HTML et CSS (partie 2)
-  Programmation HTML - Sons et animations sur le Web et VRML (partie 3)
-  Les fondamentaux du webmaster (tome d'étude et formation en ligne)
-  Création de sites avec Dreamweaver CC : Introduction
-  Création de sites avec Dreamweaver CC : CSS et conteneurs
-  Création de sites avec Dreamweaver CC : Application, transitions CSS et FTP
-  Création de sites avec Dreamweaver CC : Liens et codes
-  Création de sites avec Dreamweaver CC : Formulaires et actif
-  Création de sites avec Dreamweaver CC : Bootstrap et responsive
-  AJAX
-  Les bases de données ACCESS et MySQL
-  Découvrir PHP
-  Découvrir PHP et Mysql (tome 1)
-  Découvrir PHP et Mysql (tome 2)
-  Aller plus loin avec PHP
-  Les bases de données MySQL
-  ASP (tome 1)
-  ASP (tome 2)
-  Le référencement
-  Le langage JAVA
-  WordPress - Présentation
